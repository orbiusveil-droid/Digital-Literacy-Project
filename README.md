## Project Overview
This repository contains the deliverables for the **Student Digital Ambassador Project**.  

The objective of this project is to:
- Build a strong foundation in digital literacy  
- Establish a professional online presence  
- Practice collaborative coding  
- Develop awareness of cyber safety and professional etiquette  

The tasks are divided into five core modules, ranging from creating visual awareness resources to documenting cybercrime prevention strategies.

### Module 4: Draft a Professional Email & Etiquette Guide
**Summary:**  
- Drafted:
  - Extension request email  
  - Internship inquiry email  
- Created a checklist for responsible social media usage  

**Project Report (Communication Impact):**  
- Poor communication can harm academic/professional reputation  
- Example issue:
  - Vague subject line ("Help!")  
  - Informal language ("plz give me more time")  
- Consequence:
  - Email ignore or markd as spam  
  - Missed deadlines  

**Key takeaway:**  
Always use clear subject lines and professional tone.
